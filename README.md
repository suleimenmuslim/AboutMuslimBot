🤖 AboutMuslimBot

Telegram-бот-портфолио, созданный на Python.

📖 О проекте
boutMuslimBot — это персональный Telegram-бот, который представляет информацию обо мне, моих достижениях, проектах, интересах и пути в IT. Также бот включает ИИ-ассистента на базе Gemini AI, который может отвечать на вопросы пользователей.

 
🎯 Что делает проект?
Бот позволяет:

- 👤 Узнать информацию обо мне
- 🎯 Познакомиться с моими целями
- 🖥️ Изучить мой путь в IT
- 📈 Посмотреть прогресс обучения
- 👟 Узнать о моих увлечениях
- 🏆 Ознакомиться с достижениями и проектами
- ⭐ Оценить проекты
- 🤖 Пообщаться с ИИ-ассистентом
- 📞 Получить контактную информацию

💡 Почему проект полезен?
Проект позволяет удобно представить портфолио в интерактивном формате.
Вместо обычного документа или презентации пользователь может взаимодействовать с ботом, просматривать проекты, изучать достижения и получать ответы на вопросы через встроенный искусственный интеллект.
Проект также демонстрирует навыки разработки Telegram-ботов, работы с API и интеграции ИИ.


🚀 Как начать работу с проектом?
1. Клонируйте репозиторий
git clone [https://github.com/USERNAME/AboutMuslimBot.git](https://github.com/suleimenmuslim/AboutMuslimBot.git)

2. Установите зависимости
pip install -r requirements.txt

3.Создайте файл config.py
token_bot = "YOUR_BOT_TOKEN"
Gemini_key = "YOUR_GEMINI_API_KEY"

4. Запустите бота
python bot.py

Для запуска в режиме отладки (вывод логов действий в консоль): python bot.py debug

🛠 Используемые технологии:
-Python
-pyTelegramBotAPI
-Gemini AI API (google-genai)
-Telegram Bot API
-Regular Expressions (re)
